# 🧠 SmartCare

O **SmartCare** é um sistema de monitoramento para cuidados com idosos, utilizando sensores para detectar temperatura corporal, 
proximidade e movimento. Ideal para uso doméstico, com foco em bem-estar e prevenção.

## 💡 Funcionalidades

- 🌡️ Medição de temperatura corporal **sem contato** (sensor MLX90614)  
- 🚶‍♀️ Detecção de **movimento** (sensor PIR HC-SR501)  
- 📏 Detecção de **proximidade/distância** (sensor ultrassônico HC-SR04)  
- ☁️ Envio de dados em tempo real para o **Adafruit IO**  
- 🖥️ Dashboard online para visualização dos dados monitorados

## 🧰 Tecnologias utilizadas

- **ESP32**  
- **Arduino IDE**  
- **Adafruit IO** (para o dashboard e visualização online)  
- **Sensores**: MLX90614, PIR HC-SR501, HC-SR04  
- **Linguagem:** C/C++ (Arduino)

## 🌐 Aplicação

Este projeto é voltado para:  
- Monitoramento remoto de idosos  
- Projetos acadêmicos de IoT com foco em saúde  
- Ambientes onde é necessário detectar presença, temperatura e proximidade
